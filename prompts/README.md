# Prompts

Working prompt files for Bestie's AI personality and per-moment explanations.

Bestie is **warm, funny, encouraging, never sarcastic, never condescending**.
Responses are **conversational, beginner-friendly, and under 35 words**.

## Suggested files (to add)

- `system.md` — the core Bestie personality + rules.
- `moment_goal.md`, `moment_penalty.md`, `moment_var.md`, … — per-moment templates.
- `what_should_i_say.md` — suggestion prompt.

See [`docs/07_Prompt_Engineering`](../docs/07_Prompt_Engineering) for the strategy behind these.
